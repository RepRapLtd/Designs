♠Welcome to the world of Frank™♠

This is a site where you are able to control the little robot from the URL!

After you have entered the IP address of 192.168.1.120, you can then command him by doing e.g, "192.168.1.118/F/50" which will make Frank move forward by 50 spokes.

These are the commands that he can do:

  • F = Forward
  • B = Backwards
  • C = Turn Clockwise
  • A = Turn Anti-Clockwise
  • L = Makes the left wheel go forward
  • l = Makes the left wheel go backward
  • R = Makes the right wheel go forward
  • r = Makes the right wheel go backward
  • c = Calibrate the left and right wheel
  • d = Calculate the distance infront of it using the LIDAR
  • w = Calibrate the motors and spokes
  • v = The current voltage
  • i = The amount of light detected by the sensor
  • g = Reset the motors to 255
  • S = Set the speed of the motors
